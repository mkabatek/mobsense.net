Simple Mobile Data

MobSense is a data collection and visualization framework for mobile devices.





Mobile

Collect data from your mobile phone easily. Enumerate sensors on your device, choose which data to visualize and save.

Sensing

Collect accelerometer, magnetometer, and gyroscope data using the MobSense mobile app, or embed custom code in your own application.

Software

Download saved data for offline processing using software of your choice e.g. python, MATLAB, etc.
